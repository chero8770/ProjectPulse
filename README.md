# ProjectPulse 🚀

## Project Overview
**ProjectPulse** is a centralized hub for college students to find teammates for hackathons and final year projects. Our smart matchmaking algorithm connects project leads with students based on verified skill sets, ensuring every great idea finds the perfect team.

**Tagline:** "Find your dream team, build the future."

## ✨ Features
- ✅ Smart skill-based matching algorithm
- ✅ Verified student profiles with portfolios
- ✅ Responsive design (Mobile-first)
- ✅ Form validation & submission handling
- ✅ Smooth animations & modern UI
- ✅ Sticky navigation with mobile hamburger menu

## 🛠️ Tech Stack
- **React** (18+) - Component-based architecture
- **Tailwind CSS** - Modern, responsive styling
- **JavaScript ES6+** - Interactive
